# SmartLife Solutions - Home Automation Website

A modern, responsive website for SmartLife Solutions Pvt. Ltd., a leading home automation company based in Hyderabad, India. Built with Next.js 14, TypeScript, and Tailwind CSS.

## 🏠 About SmartLife Solutions

SmartLife Solutions Pvt. Ltd. is a Hyderabad-based home automation company that specializes in transforming homes with cutting-edge smart technology solutions. We provide comprehensive home automation services including smart lighting, security systems, climate control, and entertainment systems.

## 🚀 Features

- **Modern Design**: Clean, professional design with responsive layout
- **Fast Performance**: Built with Next.js 14 and optimized for speed
- **TypeScript**: Full type safety and better development experience
- **Responsive**: Mobile-first design that works on all devices
- **SEO Optimized**: Proper meta tags and structured data
- **Accessible**: Built with accessibility best practices

## 📱 Pages

- **Home** (`/`) - Hero section with company overview and key features
- **About** (`/about`) - Company story, mission, and values
- **Solutions** (`/solutions`) - Smart home solutions and services
- **Case Studies** (`/case-studies`) - Real customer success stories
- **Contact** (`/contact`) - Contact information and inquiry form
- **Request Demo** (`/request-demo`) - Demo scheduling form
- **Blog** (`/blog`) - Smart home tips and industry insights

## 🛠 Tech Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Components**: React functional components with hooks
- **Icons**: Emoji icons for visual appeal
- **Fonts**: Geist Sans and Geist Mono

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd smartlife-solutions
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the website.

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

## 🎨 Customization

### Colors
The website uses a blue color scheme that can be customized in the Tailwind CSS classes:
- Primary: `blue-600`
- Secondary: `gray-900`
- Background: `gray-50`

### Content
All content is currently placeholder text. Update the following:
- Company contact information in `components/Footer.tsx`
- Phone numbers and email addresses
- Actual case study data
- Blog posts with real content

### Images
Replace emoji icons with actual images:
- Add company logo
- Product images
- Case study photos
- Team photos
